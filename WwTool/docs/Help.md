## 一、设置游戏安装目录及语言

### 1.1 设置游戏安装目录

自动读取抽卡日志文件的功能依赖于游戏安装目录。请按照以下步骤进行初始化设置：

1. 启动 **WwTool.exe**。
2. 在左侧导航栏中选择 **设置**。
3. 找到 **常规游戏设置** 下的 **游戏安装目录**。
4. 点击右侧的 **选择路径** 按钮，在弹出的文件夹选择框中，选择鸣潮的安装主目录。
   - **定位游戏安装目录**
     - 通常为游戏启动器所在的目录（例如 `D:\Wuthering Waves`），或者直接是包含游戏核心程序的目录（例如 `D:\Wuthering Waves\Wuthering Waves Game`）。
5. 点击页面右下角的 **保存设置** 按钮。如果弹出“设置已保存！”的提示，即代表配置成功。

### 1.2 设置显示语言

如果您需要更改软件的显示语言：

1. 在左侧导航栏中选择 **设置**。
2. 找到 **界面设置** 下的 **显示语言**。
3. 在下拉菜单中选择您偏好的语言（例如：简体中文、English、日本語）。
4. 点击页面右下角的 **保存设置** 按钮保存生效。

## 二、账号登录与数据同步

> 本工具需要通过邮箱账号来获取相应的 Oauth 授权码。

### 1. 账号登录

1. 切换至 **首页** 导航。
2. 点击顶部的 **添加账号**按钮。
3. 在弹出的登录框中输入 **邮箱** 与 **密码**，然后点击 **登录**。

### 2. 安全验证风控提示

如果触发安全验证，请在弹出的网页窗口中完成验证，验证通过后，软件将自动完成登录和授权码保存流程。

### 3. 同步角色及探索度数据

- 登录成功后会自动刷新一次角色列表，选择想看的UID然后点击获取数据即可。
- 获取的授权码会加密保存在 `./Local/Data/LocalData.db`中

## 三、数据看板功能介绍

程序会分为多个板块展示账号信息：

### 1. 首页 (总览)

展示账号基本状态及资源恢复情况：

- **基本信息**：UID、昵称、等级、索拉等级、活跃天数、创建日期及服务器大区。
- **实时体力**：结晶玻片（体力）当前值、上限以及回满所需的剩余时间。
- **结晶单质**：溢出体力积累情况。
- **先约电台**：本期电台等级、本周电台经验及经验上限。
- **周本统计**：本周已完成的周本奖励次数。
- **活跃度**：今日活跃度完成情况。

### 2. 探索数据

从大世界同步而来的探索收集情况：

- **声匣收集进度**：展示当前背包?/世界已收集?的声匣总数。
- **奇藏箱 (Boxes)**：分别统计大世界中的 **朴素奇藏箱**、**基准奇藏箱**、**精密奇藏箱** 和 **辉光奇藏箱** 的已开启数量。
- **潮汐之遗** 的统计数据。

### 3. 摩托数据

载具（摩托车）的 DIY 及音乐收集情况：

- **载具等级与经验**：展示摩托等级以及当前升级所需经验条。
- **外观解锁统计**：已解锁的皮肤数量、贴纸数量、挂饰数量和车架数量。
- **当前装备**：显示当前正在装备的皮肤及品质。
- **皮肤 / 贴纸 / 挂饰 / 车架 / 专辑分类页**：
  - 点击对应的标签页可以查看具体解锁的每一项道具名称、品质、安装部位等。
  - **专辑**页展示当前解锁的车载音乐专辑详情及解锁曲目进度条。

## 四、抽卡历史记录获取与分析

### 1. 游戏内前置操作

由于鸣潮客户端将抽卡 API 所需的请求体数据会以临时日志形式存储于本地，因此在同步前必须进行以下操作：

1. 启动鸣潮。
2. 打开游戏内的 **唤取**（抽卡）页面。
3. 点击 **历史记录** 按钮，在游戏内至少成功翻看一页抽卡历史记录。

### 2. 自动导入链接 (推荐)

1. 在 WwTool 侧边栏选择 **抽卡统计**。
2. 在 **设置页面** 配置好游戏安装目录后，直接点击 **自动导入链接** 按钮即可。

### 3. 手动导入链接

如若自动导入提示失败，可尝试以下方法手动填入链接：

1. 找到 `Client.log` 并用文本编辑器打开。
   > 通常在 `Wuthering Waves Game\Client\Saved\Logs\Client.log`目录下
2. 在文件中搜索 `gacha/record/query` 或 `https://gmserver-api.aki-game2.net`。
3. 将包含该地址的完整 URL 复制出来。
4. 粘贴到 WwTool 的 **导入链接** 输入框中，点击 **手动导入链接** 按钮。

### 4. 数据拉取与统计分析

1. 确认解析出正确的 UID 并在下拉框中选中后，点击 **获取云端数据** 按钮。
   > 由于官方只保留6个月的抽卡记录，所以每次拉取仅能获取到六个月前的数据。
2. 拉取成功后，程序会展示各卡池的抽卡记录统计信息：
   - **总览**：总抽数、总星声花费、总出金数、小保底不歪次数、歪卡次数及不歪率。
   - **平均出金抽数**：计算平均多少抽能够获得五星角色/武器。
   - **出金历史及已垫抽数**：清晰展示出金的具体角色/武器名称、出金时所用的保底抽数，以及当前卡池已累计（已垫）的抽数。

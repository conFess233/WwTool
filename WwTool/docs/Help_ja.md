## 一、ゲームのインストールディレクトリと言語設定

### 1.1 ゲームのインストールディレクトリ設定

ログファイルからの集音データ自動取得機能は、ゲームのインストールディレクトリ設定に依存します。以下の手順に従って初期設定を行ってください：

1. **WwTool.exe** を起動します。
2. 左側のナビゲーションバーで **设置（Settings）** を選択します。
3. **常规游戏设置（ゲームの基本設定）** の下にある **游戏安装目录（ゲームのインストールディレクトリ）** を探します。
4. 右側にある **选择路径（フォルダ選択）** ボタンをクリックし、ポップアップするフォルダ選択画面で『鳴潮』のメインインストールディレクトリを選択します。
   - **インストールディレクトリの確認方法**：
     - 通常はゲームランチャーが存在するフォルダ（例：`D:\Wuthering Waves`）、あるいはゲームの実行ファイルが含まれるフォルダ（例：`D:\Wuthering Waves\Wuthering Waves Game`）です。
5. 画面右下の **保存设置（設定保存）** ボタンをクリックします。「设置已保存！（設定を保存しました！）」と表示されれば設定完了です。

### 1.2 表示言語の切り替え

ツールの表示言語を切り替えるには、以下の手順に従ってください：

1. 左側のナビゲーションバーで **设置（Settings）** を選択します。
2. **界面设置（インターフェース設定）** の下にある **显示语言（表示言語）** を探します。
3. ドロップダウンリストから、お好みの言語（日本語、English、简体中文など）を選択します。
4. 画面右下の **保存设置（設定保存）** ボタンをクリックして適用します。

## 二、アカウントログインとデータ同期

> 本ツールは、KURO GAMESアカウントのメールアドレスを使用して、対応する Oauth 認証コードを取得します。

### 1. アカウントログイン

1. **首页（ホーム）** 画面に切り替えます。
2. 画面上部の **添加账号（アカウント追加）** ボタンをクリックします。
3. ポップアップしたログイン画面で **メールアドレス** と **パスワード** を入力し、**登录（ログイン）** をクリックします。

### 2. セキュリティ検証（パズル認証）

ログイン時、セキュリティ検証（パズル認証）が発生した場合は、自動的に開くブラウザのウィンドウでスライドパズルを完了させてください。パズル認証の完了後、ツールは自動的にログイン処理を完了し、認証トークンを保存します。

### 3. 共鳴者およびエリア探索データの同期

- ログイン成功後、キャラクターリストが自動的に更新されます。表示したい UID を選択し、**获取云端数据（データ取得）** をクリックしてください。
- 取得した認証コードは暗号化され、`./Local/Data/LocalData.db` に保存されます。

## 三、ダッシュボード機能の紹介

本ツールは、アカウント情報を以下のセクションに分けて表示します：

### 1. ホーム（総覧）

アカウントの基本ステータスとスタミナ等の回復状況を表示します：

- **基本情報**：UID、ニックネーム、レベル、ソラランク、アクティブ日数、アカウント作成日、サーバー地域。
- **スタミナ（結晶玻片）**：現在のスタミナ値、上限値、満タンになるまでの残り時間。
- **結晶単質**：溢出したスタミナの蓄積状況。
- **先約ラジオ（バトルパス）**：今期のラジオレベル、今週獲得した経験値、および週上限値。
- **週ボス（周本）統計**：今週の週ボス報酬受取回数。
- **活躍度**：本日のデイリー活躍度の達成状況。

### 2. エリア探索進捗

ゲーム内の探索・収集進捗を表示します：

- **ソナーの箱（声匣）回収数**：これまでに収集した声匣の合計数。
- **宝箱（奇藏箱）**：開封した **簡易宝箱**、**標準宝箱**、**精密宝箱**、**豪華宝箱** のそれぞれの数。
- **潮汐の遺**：開封した潮汐の遺の統計データ。

### 3. 乗り物（バイク）データ

バイクのカスタマイズや車載音楽の収集進捗を表示します：

- **バイクレベルと経験値**：バイクのレベルと、次のレベルまでに必要な経験値バー。
- **外見解放数**：アンロック済みのスキン、ステッカー、ペンダント、フレームの数。
- **装備中のスキン**：現在バイクに適用されているスキンと、そのレアリティ品質。
- **スキン / ステッカー / ペンダント / フレーム / アルバムの各タブ**：
  - 各タブをクリックすることで、アンロック済みのアイテムの名称、品質、取付部位などの詳細を確認できます。
  - **アルバム**タブでは、車載音楽アルバムのアンロック情報や収録曲の進捗状況を確認できます。

## 四、集音履歴（ガチャ履歴）の取得と分析

### 1. ゲーム内での事前操作

『鳴潮』クライアントは集音 API に必要なリクエストデータをローカルの一時ログに保存するため、同期を行う前に以下の手順を完了する必要があります：

1. 『鳴潮』を起動します。
2. ゲーム内の **集音（ガチャ）** 画面を開きます。
3. **履歴（集音履歴）** ボタンをクリックし、少なくとも1ページ分以上の履歴を確認します。

### 2. 自动导入链接 (推奨)

1. WwTool のサイドバーで **抽卡统计（集音統計）** を選択します。
2. **设置（設定）** 画面でゲームのインストールディレクトリを設定した後、**自动导入链接（リンク自動導入）** ボタンをクリックします。

### 3. 链接手动导入

自動導入に失敗した場合は、以下の手順で手動でリンクをコピーして貼り付けてください：

1. `Client.log` ファイルを探し、テキストエディタで開きます。
   > ※ 通常は `Wuthering Waves Game\Client\Saved\Logs\Client.log` にあります。
2. ログファイル内で `gacha/record/query` または `https://gmserver-api.aki-game2.net` を検索します。
3. 該当のURLリンク（クエリパラメータを含む全体）をコピーします。
4. コピーしたURLを WwTool の **导入链接（リンク導入）** ボックスに貼り付け、**手动导入链接（手動導入）** をクリックします。

### 4. データの取得と統計分析

1. 解析された UID が正しいことを確認してプルダウンメニューで選択し、**获取云端数据（データ取得）** をクリックします。
   > ※ 公式サーバーのガチャ履歴保存期間は6ヶ月間のみのため、取得できるデータは過去6ヶ月間のものとなります。
2. 同期完了後、各ガチャプールごとの統計情報が表示されます：
   - **総覧**：総集音数、消費星声数、星5獲得数、すり抜けなし（50/50勝利）回数、すり抜け回数、およびその確率。
   - **星5平均集音数**：星5の共鳴者または武器を獲得するまでに引いた平均ガチャ数。
   - **獲得履歴と仮天井進捗**：獲得した星5アイテムの名称、獲得時の天井カウント、および現在の各ガチャプールにおける次回天井までの仮天井（已垫）カウントを表示します。
